# Frontend

This is the frontend of a full stack blog site built with Next.js, Graph QL and Strapi, designed with Material UI and deployed using Heroku, Amazon S3 and Vercel.

The frontend is delivered using;

- Material UI
- Next.js
- Apollo Graph QL
- Vercel
